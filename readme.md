# Grouchy

## Description

```
Open source, web based, svg editor.
```

Try [Testy](https://ionutrogojan.github.io/testy/) <--

Pixel art version; Try [Grouchy](https://ionutrogojan.github.io/grouchy/) <--

```ini
> v0.1 - initial release;
```

## Manual
```ini
0. [ N ] = canvas setup;
1. [ V / 🖱️ ] = move tool;
2. [ A / 🔧 ] = edit tool;
3. [ P / 🖋️ ] = pen tool;
4. [ M / 🧊 ] = rectangle tool;
5. [ O / 🧶 ] = elipse tool;
6. [ T / 📜 ] = text tool;
7. [ I / 👁️ ] = colour picker tool;
8. [ H / 🖐️ ] = view tool;
9. [ Z / 🔍 ] = zoom tool;
```
